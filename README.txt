Requisitos para la ejecucion del proyecto:
- Tener instalado Git
- Tener instalado NodeJS
- Tener instalado y corriendo una base de datos MySQL

Pasos para descargar el proyecto e implementarlo:
1- Descargar los archivos del proyecto del repositorio git: https://github.com/fmociulsky/restaurant.git
2- Abrir la terminal y ubicarse en la carpeta raiz del proyecto descargado.
3- Ejecutar "node install" para descargar los modulos de las dependencias
4- BD: En la carpeta BDScripts encontraran el archivo "restaurant.sql" para importar la base de datos y las tablas necesarias.
5- Iniciar el proyecto en la terminal ejecutando "node app.js"
6- En el archivo restaurant.yaml encontraran la definicion de todos los endpoints disponibles en el proyecto.
